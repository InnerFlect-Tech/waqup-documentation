# waQup Documentation

This directory contains the complete HTML documentation for waQup, optimized for GitHub Pages hosting.

## Structure

### Main Documents (7)
- `index.html` - Main documentation landing page
- `product-constitution.html` - Core identity and principles
- `scientific-foundations.html` - Research basis and psychological grounding
- `conversational-and-ritual-system.html` - How humans interact with the platform
- `ai-voice-and-ethics.html` - Voice guidelines and ethical principles
- `system-architecture.html` - System design, APIs, and infrastructure
- `value-and-growth-economy.html` - Credits, tokens, and economic model
- `roadmap-and-releases.html` - Development timeline and releases

### Subdirectories
Organized content sections for detailed documentation:
- `conversational-and-ritual-system/` - Detailed conversational system documentation
- `system-architecture/` - Detailed architecture documentation
- `value-and-growth-economy/` - Detailed economic model documentation
- `roadmap-and-releases/` - Detailed roadmap documentation

## Usage

### Viewing Locally
Open `index.html` in a web browser to view the documentation.

### GitHub Pages
This directory is configured to be served as the root of your GitHub Pages site. Configure GitHub Pages to use the `/docs` folder as the source.

## Features

All HTML files are self-contained and include:
- Complete styling and responsive design
- Navigation sidebar
- Mermaid diagram support (via CDN)
- No external dependencies required

## Organization

This is a **pure HTML** documentation system. All content is integrated into self-contained HTML files. No build system, dependencies, or source files are required.
