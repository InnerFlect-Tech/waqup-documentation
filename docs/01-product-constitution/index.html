Core Identity

waQup is a voice-first ritual creation platform that enables intentional self-authorship through three depths of subconscious engagement:

  1. Affirmations → cognitive re-patterning
  2. Guided Meditations → state induction
  3. Rituals → identity encoding

waQup is not a meditation app, therapy tool, or social network. It is a tool for encoding identity-level change through personalized, voice-guided practices.

Core Principles

1. Practice is Free, Creation is Sacred

  • Unlimited replay of all created content
  • Credits consumed only for creation/transformation
  • No subscription pressure or "use it or lose it" mechanics
  • Value is created at the moment of encoding, not passive consumption

2. No Gamification

  • No streaks, badges, or achievements
  • No comparison with other users
  • No external rewards or punishments
  • Users return for intrinsic value, not external motivation

3. User Autonomy

  • Users control their practice
  • No pressure to return
  • No consequences for not practicing
  • Easy exit (export, delete account)
  • Respect for user agency

4. Voice-First Design

  • All core interactions via voice
  • Text and visual interfaces are secondary
  • Audio is primary delivery mechanism
  • Supports nervous system regulation through voice prosody

5. Conversation Over Forms

  • All content creation happens through dialogue
  • Natural, adaptive conversation flow
  • No static forms or configuration screens
  • Context-aware personalization

6. Three Depths, Not Interchangeable

  • Affirmations, Guided Meditations, and Rituals serve different psychological purposes
  • They are not interchangeable
  • Users progress naturally from shallow → deep
  • Each type has distinct structure and purpose

What waQup Does

  • Creates personalized voice-guided content (affirmations, meditations, rituals)
  • Supports embodied practice through conversation
  • Provides marketplace for creator content
  • Enables habit formation through return loops
  • Recognizes contribution through tokens (non-speculative)
  • Manages AI resource consumption through credits
  • Offers unlimited replay of all created content

What waQup Never Does

  • Gamifies practice (no streaks, badges, leaderboards)
  • Creates dependency (no withdrawal mechanics, no FOMO)
  • Manipulates users (no urgency, pressure, dark patterns)
  • Claims medical benefits (not a therapy tool, not medical advice)
  • Tracks users invasively (minimal data collection, privacy-first)
  • Creates social comparison (not a social network, no public profiles)
  • Forces engagement (no reminders, no pressure to return)
  • Expires credits (credits never expire, no FOMO)

Boundaries

What waQup Is

  • A tool for self-authorship and identity encoding
  • A platform for intentional creation
  • A marketplace for creator content
  • A resource management system for AI generation

What waQup Is Not

  • A therapy tool (does not claim medical benefits)
  • A social network (no social comparison, no public profiles)
  • A meditation app (broader than meditation, includes identity work)
  • A subscription service (credits-based, not time-based)
  • A gamification platform (no rewards, streaks, or achievements)

Invariants (Non-Negotiable)

These principles cannot change without fundamentally altering waQup's identity:

  1. Practice is free (unlimited replay)
  2. No gamification (intrinsic motivation only)
  3. User autonomy (no manipulation, no pressure)
  4. Voice-first (audio is primary delivery)
  5. Three depths (affirmations, meditations, rituals are distinct)
  6. Conversation-based creation (no forms)
  7. Credits never expire (no FOMO)
  8. Ethical data handling (minimal collection, user control)

Scope

In Scope

  • Voice-guided content creation (affirmations, meditations, rituals)
  • Personalized audio generation
  • Content library and playback
  • Creator marketplace
  • Credit-based resource management
  • Token/reward system (non-speculative)
  • User autonomy and privacy

Out of Scope

  • Medical/therapy claims
  • Social networking features
  • Gamification mechanics
  • Subscription-based pricing
  • Time-based restrictions
  • Public user profiles
  • Social comparison features

Evolution Principles

When considering new features or changes:

  1. Does it respect user autonomy? (no manipulation)
  2. Does it maintain anti-gamification? (no external rewards)
  3. Does it preserve voice-first design? (audio is primary)
  4. Does it honor the three depths? (types remain distinct)
  5. Does it align with ethical principles? (privacy, consent, transparency)

If any answer is "no," the feature cannot be added without violating waQup's constitution.

Summary

waQup is a voice-first ritual creation platform that enables intentional self-authorship through three depths of subconscious engagement. It prioritizes user autonomy, avoids gamification, and respects psychological layers. Practice is free, creation is sacred, and users maintain full control over their journey.
